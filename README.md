## Task 05 - Data Science Internship at Prodigy Infotech

### Objective
Analyze traffic accident data to identify patterns related to road conditions, weather, and time of day, and visualize accident hotspots.

### Dataset
US Accidents Dataset (US_Accidents_March23.csv)

### Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded the dataset using pandas.
2. Explored the dataset to understand its structure and features.
3. Cleaned and processed the data.
4. Analyzed accidents based on time, weather conditions, and location.
5. Visualized accident patterns using charts and graphs.

### Output
- Visualizations showing accident trends.
- Insights into factors affecting road accidents.
- Identification of accident hotspots.

### Conclusion
The analysis helps understand the major factors contributing to road accidents and provides insights that can help improve road safety and traffic management.
